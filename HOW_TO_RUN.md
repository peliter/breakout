# 如何啟動遊戲

當您直接在瀏覽器中開啟 `index.html` 檔案時，可能會遇到跨來源資源共享 (CORS) 錯誤，導致遊戲腳本無法載入。為了解決這個問題，您需要透過本地伺服器來提供這些檔案。

以下是使用 Python 內建 `http.server` 模組啟動本地伺服器並執行遊戲的步驟：

1.  **開啟終端機 (Terminal)**
2.  **導航到專案根目錄**
    使用 `cd` 命令進入您的專案目錄：
    ```bash
    cd /Volumes/Windows/macbookairm2/game_breakout
    ```
3.  **啟動本地伺服器**
    在終端機中執行以下命令來啟動一個簡單的 HTTP 伺服器，預設使用 8000 埠：
    ```bash
    python -m http.server 8000
    ```
    如果您沒有安裝 Python，或者遇到權限問題，您可能需要安裝 Python 或使用其他本地伺服器解決方案。

4.  **在瀏覽器中開啟遊戲**
    伺服器啟動後，開啟您的網頁瀏覽器，並輸入以下網址來訪問遊戲：
    ```
    http://localhost:8000/index.html
    ```

現在，遊戲應該可以正常載入並執行，而不會出現 CORS 錯誤。
